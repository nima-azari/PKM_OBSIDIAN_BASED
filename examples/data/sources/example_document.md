# Example Document

This is an example markdown file to demonstrate how the PKM system works.

## About This Tool

This Personal Knowledge Management (PKM) system allows you to:
- Store and organize documents
- Build knowledge graphs
- Query your knowledge with AI
- Generate synthesis articles

## Getting Started

1. Add your documents to `examples/data/sources/`
2. Run `python build_graph.py`
3. Run `python server.py` to start chatting
